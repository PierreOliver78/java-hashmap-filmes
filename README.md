# 🎬 Java HashMap - Movie Categorization

Aplicação em Java que organiza filmes por gênero utilizando HashMap e List, demonstrando conceitos fundamentais de orientação a objetos e estruturas de dados.

---

## 🚀 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar:

- Programação orientada a objetos (POO)
- Uso de List (ArrayList)
- Uso de HashMap
- Agrupamento de dados por categoria
- Estruturas de repetição (for-each)

---

## 🛠 Tecnologias

- Java 11
- Eclipse IDE
- Git & GitHub

---

## 📂 Estrutura

- model → Classe Filme
- main → Classe executável TesteFilmes

---

## ⚙️ Como funciona

- Criação de objetos Filme
- Armazenamento em uma lista
- Agrupamento por gênero usando HashMap
- Impressão organizada no console

---

## 💻 Exemplo de saída

```
***** Ficção *****
Matrix - Nota: 9.8
Mad Max - Nota: 8.9

***** Drama *****
Titanic - Nota: 9.5
A Teoria de Tudo - Nota: 7.7
```


---

## 🌟 Destaques

- Uso prático de HashMap para categorização
- Código limpo e organizado
- Projeto focado em fundamentos essenciais de Java
